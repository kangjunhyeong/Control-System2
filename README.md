(a) $$y(t) = x_1(t), \frac{dy(t)}{dt} = x_2(t) $$  
(b) $$M\frac{d^2y(t)}{dt^2} + b\frac{dy(t)}{dt} + ky(t) = F(t)\$$  
$$\frac{dx_2(t)}{dt} = \frac{F(t) - bx_2(t) - kx_1(t)}{M}\$$  
(c)
$$\frac{dx_1(t)}{dt} = x_2(t)\$$  
$$\frac{dx_2(t)}{dt} = \frac{F(t) - bx_2(t) - kx_1(t)}{M}\$$
